This application is designed for Windows Vista/7/8/8.1/10
and allows you to monitor Embarcadero Delphi classes.

The following web site contains useful information about memory leak detection.

[http://delphi.about.com/od/kbcurt/a/dynamiccreation.htm]
[http://delphi.about.com/od/adptips2005/qt/nilselfapp.htm]

Installation :

- Execute the application setup executable.
- Launch the 'Delphi Class Monitor' program from
  the 'Devonic' program group.
