[https://cycocrew.github.io/devonic/]

This application is designed for Windows Vista/7/8/8.1/10/11
and allows you to manage airline flights. This includes management of flights, airlines,
airports and distances.

The following web sites can be used to calculate distances between two airports.

[http://www.convertunits.com/distance/]
[http://www.webflyer.com/travel/milemarker/]

Installation :

- Execute the application setup executable.
- Launch the 'Flight Tracker' program from the 'Devonic' program group.
