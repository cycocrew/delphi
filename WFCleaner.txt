[https://cycocrew.github.io/devonic/]

This application is designed for Windows Vista/7/8/8.1/10/11
and allows you to clean selected folder content.

This application requires Administrator privileges.

Installation :

- Execute the application setup executable.
- Launch the 'Windows Folder Cleaner' program from the 'Devonic' program group.
