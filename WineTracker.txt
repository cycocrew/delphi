[https://cycocrew.github.io/devonic/]

This application is designed for Windows Vista/7/8/8.1/10/11
and allows you to manage wine cellars. This includes management of wines, producers and
appellations.

Installation :

- Execute the application setup executable.
- Launch the 'Wine Tracker' program from the 'Devonic' program group.
